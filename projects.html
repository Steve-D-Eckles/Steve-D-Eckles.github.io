This Site!

This site was created from the ground up using HTML, CSS, blood, sweat, and tears. It was not only a chance to apply the skills I gained in the first few weeks of the CSET program at Thaddeus Stevens but also a sandbox for experimenting with new tools to fit my design.

This project was version controlled with Git and has a repo available for public perusal on GitHub.

Fragile

Fragile is a browser-based JavaScript puzzle game created by Brian Burchett and me as the final project in our first semester of the CSET program. It was planned and implemented using the Agile process and a centralized Git workflow. We planned requirements, created user stories and assigned them points, and had daily standups with our instructor to discuss current plans and roadblocks and keep track of our burndown rate.

There is, of course, a Git repo on GitHub

Mountain of Madness

Mountain of Madness is a text-based adventure game written in Python. It served as an introduction to Object-Oriented Programming and was based on a framework laid out in Zed Shaw's Learn Python the Hard Way.

Other skills that were practiced in the development of this game include splitting a program into modules, following proper naming conventions, and improving usage of comments and docstrings.
